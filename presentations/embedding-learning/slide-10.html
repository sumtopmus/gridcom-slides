Applications in Production

Semantic Search & RAG

Embed corpus into vector DB, query by meaning, retrieve nearest docs, feed to LLM for grounded generation. The dominant embedding application today.

Recommendation Systems

YouTube, Spotify, Amazon: two-tower models generate user & item embeddings. ANN search over millions of candidates in milliseconds.

Face Recognition

FaceNet: CNN + triplet loss maps faces to 128-d space. Same person clusters tightly regardless of angle, lighting, or age.

Anomaly & Fraud Detection

Normal behavior clusters tightly in embedding space. Anomalies appear as distant outliers — adapts to new patterns without hand-crafted rules.

Drug Discovery

Molecular embeddings via GNNs enable similarity search over billions of compounds. Promising candidates found in hours, not months.

Code Intelligence

Copilot retrieval, semantic code search, clone detection. Syntactically different but functionally equivalent code maps to nearby vectors.
