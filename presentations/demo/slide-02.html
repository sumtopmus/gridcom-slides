Features

Everything you need,
nothing you don't

🖼️
iframe isolation

Each slide is a sandboxed iframe. Full CSS isolation, arbitrary scripts, zero bleed between slides.

⌨️
Keyboard-first navigation

Arrow keys, Space, F for fullscreen, G for grid overview. Everything reachable without a mouse.

🔗
Shareable URLs

Every slide has a unique URL. Navigate directly to any slide with #/demo/2.

✨
Lifecycle hooks

Slides receive slideEnter and slideExit events via postMessage. Trigger animations on cue.
